# Paramsnode - Scanner Parameters

# Apa itu ParamsNode?
ParamsNode ialah alat Berbasis Node.js yang dibuat untuk melakukan Scanner Terhadap Endpoint (Parameter) secara Mudah

# Bagaimana cara Menggunakannya?
***Install Package***
*Update Package*
```bash
pkg update -y
```
*Upgrade Package*
```bash
pkg upgrade -y
```
*Install Node.js*
```bash
pkg install nodejs-lts -y
```
*Install Git*
```bash
pkg install git -y
```
# GitClone dan Menjalankannya
*Clone Repository*
```bash
git clone https://github.com/NekomonHub/paramsnode
```
*Change Directory*
```bash
cd paramsnode
```
# Install Npm dan Running
```bash
npm install
```
*Start Script / Run Script*
```bash
node param.js
```

# Bagaimana Contoh Penggunaannya?
**Contoh Menjalankannya**
```bash
node param.js <URL>
```
**Example Usage**
```bash
node param.js https://google.com
```
